Imtahan Project Şagirdlerin imtahan neticesini sisteme əlavə etmək üçün yaradılmış ASP.NET MVC proyektdir
Projecti run eləmək ucun DOCKER mühitinə ehtiyac var
Run eləmək ucun aşağıdakı addımlar yerinə yetirilməlidir

1) Docker engine qaldirilmali
2) docker-compose file run olunmali, ve sql serverin qalxmasindan emin olunmali
3) Package manager consoleda Update-Database komandasi icra olunmali
4) dotnet run komandasi icra olunmali
